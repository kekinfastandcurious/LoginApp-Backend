# LoginApp-Backend
Login APIs develop in express+NodeJS for POC-2

## Usage Guide
* Clone from the repo
    git init
    ```sh
    git init
    git clone git@github.com:kekinfastandcurious/LoginApp-Backend.git
    git checkout -b master
    git pull origin master
    ```
* Start the App
    ```sh
    npm i
    npm start
    ```
